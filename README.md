## Implements:
- Parallel FIR filtering 
- Max Pooling Layer
- Command Line Stride Visualization 

## Usage:
1. Compile the simulator:
```bash
gcc -g *.c -o sim
```

2. Generate input shapes:
```bash
python generate_shapes.py [image_width] [square_length] [circle_radius]
```

3. Run the simulator:
```bash
# Fast mode
./sim 100 -f

# Medium mode  
./sim 100 -m

# Slow mode (default)
./sim 100 -s

# Manual step-through mode
./sim 100 --step
```
